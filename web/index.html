The System
Campaign Management
for the Uncharted
A self-hosted command center for tabletop RPG groups running homebrew worlds. Character sheets, living lore, alchemical databases, and arcane circles — all synchronized in real-time between the Dungeon Master and every player at the table.
⚔
Character Sheets
Nine attributes, ranked abilities, arcane affinities, inventory with rarity, trait modifiers — the full sheet, alive and editable.
🜁
Transmutation Circles
A function-notation magic system. ERAM, CIRCUM, CREO — compose spells like code, chain them like circuits.
◆
Runic Circuits
Input, Capacitor, Circuit, Discharge, Ground. Energy flows or the rune explodes. Build with care.
⚗
Alchemy Database
Ingredients from every biome — leaves, roots, fungi, creatures. Multi-part effects with power tiers.
☽
Living Lore
Wiki-style entries for world, factions, NPCs. Linked to character races. The story grows with every session.
⟐
Real-time Sync
WebSocket-powered. DM changes souls, player sees it instantly. No refresh needed.
